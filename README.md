# Aspect-Based Customer Review Sentiment Analyzer

This project is an NLP-based web application that analyzes mobile phone customer reviews and predicts sentiment for different product aspects such as battery, camera, display, and performance.

The application uses:
- Python
- VADER Sentiment Analysis
- Streamlit
- NLP text processing

## Features
- Sentiment Analysis (Positive / Negative / Neutral)
- Aspect Extraction
- Interactive Streamlit Dashboard
- Real-time Review Analysis

## Technologies Used
- Python
- Streamlit
- vaderSentiment
- NLP

## Example Review
Input:
Camera quality is amazing but battery drains fast.

Output:
- Aspects Found: Camera, Battery
- Sentiment: Negative
